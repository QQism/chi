===
chí
===

A restructuredtext compiler which generates HTML content for
JavsScript, ClojureScript, and Clojure

**NOTE**: This library is under development and may subject to change

Usage
-----

Document Structure

- Document `Checked`
- Section `Checked`

  + Title `Checked`
- Transition `Checked`

Document Body

- Paragraph `Checked`
- Inline markup

  + Emphasis
  + Strong emphasis
  + Interpreted text
  + Inline literals
  + Standalone hyperlink
  + External hyperlink
  + Internal cross-references
  + footnote references
  + Citation references
  + substitution references
  + Inline internal target
- Block

  + Literal block
  + Block quotes `Checked`
  + Doctest block
- Lists

  + Bullet `Checked`
  + Enumerated
  + Definitions
  + Field lists
  + Option lists
- Table

  + Grid table `Checked`
  + Simple table
- Explicit markup blocks

  + Footnote
  + Citation
  + Hyperlink target
  + Directives

    - highlight.js
    - Mathjax
    - Tables
    - Images
  + Substitution definitions
  + Comments

What is `chí`?
--------------

`chí` is a Sino-Vietnamese word. The word has mutiple meanings, depending on context, and can be referred as *annals*, or *aspiration*.
